SCHIZZO PROJECTS

FBRHC BUILDING

Project Architect: Ligia Saldaña Martorell, AIA, CAAPPR

Design Team: Atelier 66, csp

Location: Ponce, Puerto Rico

Year: 2014

Fundación Biblioteca Rafael Hernández Colon (FBRHC)

This project consists of the new facilities for the foundation of one of Puerto Rico’s former governor. The building is really two buildings that work seamlessly. One is historic in nature and the other modern. The corner building is a century old structure that was updated to accommodate the client’s program. The new annex building houses all modern amenities necessary to comply with the proposed use and current building code requirements. All historical construction techniques were used for the restoration work.
